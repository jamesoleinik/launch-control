**Launch Control · Episode 9: The Dashboard.**

Executives don't talk to agents. They look at dashboards.

This episode ships **Launch Command Center** — a generative Power Apps page that surfaces the live state of every launch from Dataverse. KPI tiles, status by milestone, blockers list, drill-down to tasks.

The interesting part isn't the page — it's *how it got there*: deployed entirely from the CLI with `pac model genpage upload`. No maker-portal click-ops. No screenshots in a doc. The page is in source control like every other artifact, and a script sets it as the default landing for the model-driven app.

If a Power Apps page is generative, source-controlled, and CLI-deployable, it's a real engineering artifact — not a configuration screenshot.
