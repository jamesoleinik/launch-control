**Launch Control · Episode 6: The Agent.**

Five episodes of substrate. Time to put a face on it.

Meet **Launch Coordinator** — a declarative Copilot Studio agent grounded on Dataverse and a knowledge base of sanitized launch playbooks. Ask it "where are we on Smart Widget Pro?" and it answers from the same tables the dashboard reads.

What makes it actually useful (vs. a glorified search box):
• It uses the **Business Skills** from Episode 2 as its decision rules
• It calls the **Custom API** from Episode 5 to get real readiness scores
• It can pivot to live **GitHub Issues** through the virtual entity from Episode 4

The declarative agent is 100% configuration, no code. Every previous episode's work compounds here.
